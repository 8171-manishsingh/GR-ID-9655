# Quiz App with Timer

A simple Quiz Application built using HTML, CSS, and JavaScript.

## Features
- Multiple choice questions
- Timer for each question
- Correct and wrong answer highlight
- Next question button
- Final score display

## Project Structure

quiz-app/
│
├── index.html
├── style.css
├── script.js
└── README.md

## How to Run

1. Download or clone the project.
2. Extract the ZIP file.
3. Open index.html in your browser.

Your quiz app will start running.

## Technologies Used

- HTML
- CSS
- JavaScript
